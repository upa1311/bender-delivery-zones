## Not data sources

Google Maps, Yandex Maps, the public Nominatim service, and the public Overpass
API are **not** import sources for this project. See
[docs/licensing.md](docs/licensing.md) and
[docs/data-sources.md](docs/data-sources.md).

## Address register

The official Moldova address register is treated as a *potential validation
source only*, with bulk-access and reuse terms currently **unverified**. It is
not imported in this stage.
